# SwitchID 用户使用指南

## 📖 产品定位

**SwitchID 是一个隐私保护和账户管理工具**，专注于：
- 🔒 **隐私保护**：防止浏览器指纹追踪，保护用户隐私
- 📁 **本地账户管理**：在本地设备上管理多个账户的登录状态
- 🛡️ **数据安全**：所有数据仅存储在用户本地设备，不上传任何服务器

> ⚠️ **重要声明**：本工具仅提供技术功能，用户需自行确保使用合规性

## 🎯 核心功能说明

### 1. 隐私保护功能
```
功能描述：
✓ 浏览器指纹伪装 - 防止网站追踪用户设备特征
✓ User-Agent 管理 - 保护浏览器身份信息
✓ Canvas 指纹保护 - 防止基于图形渲染的追踪
✓ WebGL 指纹保护 - 防止基于3D渲染的追踪

使用场景：
✓ 保护个人隐私，防止被恶意追踪
✓ 研究网络安全和隐私保护技术
✓ 测试网站的指纹检测机制
```

### 2. 本地账户管理功能
```
功能描述：
✓ 本地保存登录状态 - 将Cookie等信息存储在本地
✓ 快速切换账户 - 在不同账户间快速切换
✓ 数据导入导出 - 备份和恢复账户数据
✓ 分组管理 - 对账户进行分类管理

技术原理：
✓ 使用浏览器标准API（Chrome Storage API）
✓ 操作浏览器Cookie和本地存储
✓ 不涉及任何服务器通信
✓ 完全离线工作
```

## 🔧 详细使用教程

### 第一步：安装和初始设置

#### 1.1 安装扩展
1. 下载扩展文件
2. 打开浏览器扩展管理页面
3. 启用"开发者模式"
4. 加载已解压的扩展程序

#### 1.2 首次使用设置
1. 点击扩展图标打开弹窗
2. 阅读使用提醒和风险警告
3. 根据需要配置隐私保护选项
4. 了解基本操作流程

### 第二步：隐私保护设置

#### 2.1 指纹保护配置
```
推荐设置：
✓ Canvas指纹保护：开启（防止图形追踪）
✓ WebGL指纹保护：开启（防止3D渲染追踪）
✓ Audio指纹保护：开启（防止音频追踪）
✓ Font指纹保护：谨慎开启（可能影响网页显示）

注意事项：
⚠️ 过度的指纹伪装可能影响网站功能
⚠️ 某些网站可能检测并阻止指纹伪装
⚠️ 建议根据实际需要选择保护级别
```

#### 2.2 User-Agent 管理
```
使用方法：
1. 在设置页面选择预设的User-Agent
2. 或者输入自定义的User-Agent字符串
3. 可以为不同账户设置不同的User-Agent
4. 支持随机生成常见的User-Agent

风险提醒：
⚠️ 不常见的User-Agent可能引起网站注意
⚠️ 频繁更换User-Agent可能触发安全检查
⚠️ 建议使用主流浏览器的User-Agent
```

### 第三步：账户管理操作

#### 3.1 保存账户状态
```
操作步骤：
1. 在网站上正常登录您的账户
2. 点击扩展图标，选择"保存账户"
3. 输入账户名称（如"工作账户"、"个人账户"）
4. 选择颜色标识便于区分
5. 点击"确认保存"

保存内容：
✓ 网站Cookie信息
✓ localStorage数据
✓ sessionStorage数据
✓ 账户配置信息

重要提醒：
⚠️ 仅保存您合法拥有的账户
⚠️ 确保在完全登录状态下保存
⚠️ 定期更新过期的账户信息
```

#### 3.2 切换账户
```
操作步骤：
1. 点击扩展图标打开弹窗
2. 在账户列表中选择目标账户
3. 点击"切换"按钮
4. 等待页面自动刷新完成切换

切换过程：
1. 清除当前网站的登录状态
2. 加载目标账户的保存数据
3. 自动刷新相关页面
4. 完成账户切换

注意事项：
⚠️ 切换过程中请勿关闭浏览器
⚠️ 确保网络连接稳定
⚠️ 某些网站可能需要额外验证
```

#### 3.3 账户管理
```
管理功能：
✓ 编辑账户信息（名称、备注、标签）
✓ 删除不需要的账户
✓ 导出账户数据进行备份
✓ 导入之前备份的数据

分组功能：
✓ 创建账户分组（如"工作"、"个人"）
✓ 将账户分配到不同分组
✓ 按分组查看和管理账户
✓ 支持账户标签系统

数据管理：
✓ 定期备份重要账户数据
✓ 清理过期或无效的账户
✓ 检查账户健康状态
✓ 更新账户登录信息
```

## 📋 常见问题解答

### Q1: 这个工具是否合法？
**A**: 本工具提供的是标准的浏览器功能，技术本身是合法的。但使用方式需要符合相关法律法规和网站服务条款。用户需要自行确保使用的合规性。

### Q2: 会不会导致账户被封？
**A**: 存在这种风险。不同网站对多账户使用有不同政策，某些网站可能检测并限制此类行为。建议使用前了解网站政策，合理使用。

### Q3: 数据安全吗？
**A**: 所有数据都存储在您的本地设备上，不会上传到任何服务器。但您需要保护好自己的设备安全，建议定期备份重要数据。

### Q4: 指纹保护效果如何？
**A**: 可以有效防止常见的指纹追踪技术，但无法保证100%的保护效果。网站的追踪技术在不断演进，需要持续更新保护机制。

### Q5: 如何降低使用风险？
**A**: 
- 仔细阅读并遵守网站服务条款
- 仅管理您合法拥有的账户
- 保持合理的使用频率
- 定期备份重要数据
- 及时关注账户状态变化
